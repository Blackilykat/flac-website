--- layout: default title: Example code --- The FLAC source code has several small example programs that demonstrate how to use the libraries. The source is available on the download page, or can be checked out from git or browsed online. The examples complement the API documentation.

Currently the examples show how to encode WAV files to FLAC and vice-versa using both libFLAC and libFLAC++. Over time we'll be adding more examples.
